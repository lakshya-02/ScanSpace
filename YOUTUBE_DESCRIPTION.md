# YouTube Trailer Description

ScanSpace is a Meta Quest 3 mixed reality app that turns real-world objects into interactive 3D models.

Point the headset at an object, capture an image, crop the object, and send it to a local AI-powered Stable Fast 3D server. Within moments, ScanSpace brings the generated 3D model back into your MR space, where you can move, rotate, scale, place, remove, and save it for later.

Built with Unity, Meta Quest 3 passthrough, FastAPI, and Stable Fast 3D.

Key features:
- One-button object capture in mixed reality
- Crop-based image cleanup before generation
- AI-powered 3D model generation
- Interactive model placement, rotation, scaling, and dragging
- Local model saving and restore
- Quest-friendly feedback UI and sound effects
